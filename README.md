# Restaurant Rescue Agent

## Overview

Restaurant Rescue Agent is an AI-powered analytics copilot designed to help restaurant partners improve customer satisfaction, ratings, and order performance.

The platform uses restaurant data, customer reviews, menu insights, and business signals available through Swiggy MCP APIs to generate actionable recommendations that help restaurants grow.

---

## Problem Statement

Many restaurant owners struggle to understand:

* Why orders increase or decrease
* Which menu items perform best
* What customers like or dislike
* How ratings can be improved
* Which areas require immediate attention

Analyzing reviews and performance metrics manually is time-consuming and often leads to missed opportunities.

---

## Solution

Restaurant Rescue Agent acts as an AI business consultant that continuously analyzes restaurant data and provides clear recommendations.

Examples:

* Identify frequently mentioned customer complaints
* Detect top-performing and underperforming menu items
* Analyze customer sentiment
* Discover peak demand periods
* Suggest actionable improvements

---

## Key Features

### Review Intelligence

Summarizes customer feedback and identifies common positive and negative themes.

### Menu Performance Analysis

Highlights best-selling and underperforming menu items.

### Sentiment Analysis

Uses AI to understand customer satisfaction trends.

### Business Insights Dashboard

Provides visual analytics for restaurant performance.

### AI Copilot

Allows restaurant owners to ask business questions in natural language.

Example:

"Why did my ratings drop this week?"

"What menu items should I promote?"
